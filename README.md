# Personal Planner

여행 계획을 적는 어플리케이션

## 프로젝트 소개
 자바 스윙을 이용한 개인 여행 일정을 손쉽게 등록하고 여행 계획을 추가할 수 있으며 다른 여행자들과의 경험을 공유하고 유용한 여행 팁을 얻을 수 있는 프로그램입니다.

## Getting Started

### 라이브러리

* JTattoo v1.6.18 .jar
* ojdbc 8 .jar
* JFreeChart

### 개발환경

* DB : Oracle 11
* Server : Apache Tomcat v.9 , MyBatis

### 툴

* Eclipse
* SQL Developer
* exERD
* draw.io
* Figma
